Projet - GSB

Contexte du projet

Le projet GSB (Galaxy Swiss Bourdin) a été réalisé dans le cadre de ma deuxième année de BTS SIO. Il s'agissait de concevoir et de mettre en œuvre une infrastructure réseau sécurisée et conforme aux recommandations de l'ANSSI (Agence Nationale de la Sécurité des Systèmes d'Information) pour une entreprise fictive.

Objectifs

  • Concevoir une architecture réseau robuste et sécurisée
  • Implémenter les services essentiels (Active Directory, DHCP, DNS)
  • Mettre en œuvre des mesures de cybersécurité (pare-feu, VPN, VLAN)
  • Rédiger une documentation technique complète

Technologies utilisées

  • Virtualisation : Proxmox
  • Pare-feu : pfSense
  • Serveur : Windows Server & Linux
  • Protocoles : VLAN, Active Directory
