Projet - Homelab (Raspberry Pi & Docker)

Contexte du projet

Ce projet vise à créer un environnement de laboratoire personnel ("Homelab") à la maison, basé sur un Raspberry Pi. L'objectif est d'héberger plusieurs services auto-hébergés via Docker, accessibles de manière sécurisée via un VPN.

Composants et Services

  • Raspberry Pi : Serveur principal pour l'hébergement Docker.
  • Docker : Plateforme de conteneurisation pour déployer et gérer les services.
  • VPN (WireGuard/OpenVPN) : Pour un accès sécurisé et distant aux services internes du Homelab.
  • Nextcloud : Solution de cloud personnel pour le stockage et la synchronisation de fichiers.
  • Jellyfin : Serveur multimédia personnel pour le streaming de films, séries et musique.
  • Portainer : Interface graphique pour la gestion simplifiée des conteneurs Docker.
  • Site web personnel : Un portail centralisé pour accéder et gérer tous les services du Homelab, incluant des liens vers Nextcloud, Jellyfin, etc.

Objectifs du projet

  • Déployer Docker sur Raspberry Pi.
  • Mettre en place un serveur VPN pour un accès sécurisé.
  • Installer et configurer Nextcloud pour le stockage de fichiers.
  • Installer et configurer Jellyfin pour le streaming multimédia.
  • Utiliser Portainer pour la gestion visuelle des conteneurs.
  • Créer un site web simple pour centraliser l'accès aux services.
  • Apprendre les bonnes pratiques de sécurité et de gestion des services auto-hébergés.

Compétences développées

  • Administration système sous Linux (Raspberry Pi OS).
  • Maîtrise de Docker et Docker Compose.
  • Configuration de réseaux VPN.
  • Déploiement et gestion de services web (Nextcloud, Jellyfin).
  • Sécurité des systèmes auto-hébergés.
  • Gestion de projets personnels d'infrastructure.
